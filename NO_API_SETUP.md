# 零配置使用指南

## 简介

本应用支持**无需申请任何API Key**即可使用基础天气查询功能！通过网页爬取技术，应用可以从公开的天气网站获取数据。

## 快速开始

### 1. 直接运行

无需修改任何配置，直接运行应用：

```bash
flutter run
```

### 2. 使用说明

应用会自动：
- 使用百度天气接口获取数据
- 使用天气网作为备用数据源
- 支持全国城市天气查询
- 提供7天天气预报

## 数据源说明

### 主要数据源

1. **百度天气接口**
   - 公开接口，无需Key
   - 数据较准确
   - 支持全国城市

2. **天气网爬取**
   - 备用数据源
   - 实时网页数据
   - 自动解析天气信息

3. **360天气接口**
   - 城市搜索建议
   - 快速响应

## 配置选项

### 切换数据源

编辑 `lib/config/app_config.dart`：

```dart
// 优先使用API（需要配置Key）
static const bool useApiFirst = true;

// 仅使用网页爬取（无需Key）
static const bool useApiFirst = false;
```

### API Key配置（可选）

如果你后续申请到了API Key，可以填入：

```dart
// 高德天气API Key
static const String amapWeatherKey = '你的Key';
```

留空或保持默认将自动使用网页爬取。

## 注意事项

1. **网络要求**
   - 需要联网使用
   - 首次加载可能稍慢

2. **数据准确性**
   - 网页数据与官方API基本一致
   - 可能会有几分钟延迟

3. **稳定性**
   - 网页结构变化可能影响爬取
   - 应用会自动回退到备用源

4. **AI建议功能**
   - 需要配置AI API Key
   - 可选：Kimi / 通义千问 / DeepSeek

## 故障排除

### 无法获取天气数据

1. 检查网络连接
2. 尝试切换WiFi/移动数据
3. 重启应用
4. 检查城市名称是否正确

### 数据不准确

1. 网页数据可能有延迟
2. 尝试刷新
3. 考虑申请API Key获得更精准数据

## 升级建议

长期使用建议申请API Key：

1. **高德天气API**
   - 免费额度充足
   - 数据更稳定
   - 功能更完整

2. **Kimi AI**
   - 新用户免费额度
   - 智能建议更准确

## 技术说明

网页爬取工作原理：

1. 应用发送HTTP请求到天气网站
2. 获取网页HTML内容
3. 解析HTML提取天气数据
4. 转换为应用数据模型
5. 缓存到本地数据库

整个过程对用户透明，无需额外操作。
